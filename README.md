# Producer Gym

A web-based daily training platform for music producers. Each day, users are presented with a short reference loop and challenged to recreate it as accurately as possible.
After uploading their version, they receive instant, objective scoring and see their position on a daily leaderboard.

## Getting Started

First, run the development server:

```bash
# clone the repo
git clone https://github.com/serfoll/producer-gym

# install dependencies
npm install

# start dev server
npm run dev

# view in brower
http://localhost:300
```
